# Tune Share Hub Frontend

React JavaScript client for Tune Share Hub.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run lint`

## API

By default, Vite proxies `/api` to `http://localhost:8080`.
Set `VITE_API_BASE_URL` when the backend runs on a different origin.

See `../docs/spotify-setup.md` for Spotify Dashboard and environment variable setup.
